## [1.0.0] - 2024-XX-XX

### 新增

#### 核心功能
- 实现完整的中国象棋规则
- 支持人机对弈模式
- 支持人人对弈模式
- 支持悔棋和重做功能
- 支持提示功能（请求引擎提示）

#### 棋盘和界面
- 精美设计的棋盘界面
- 支持点击和拖拽两种走子方式
- 三种棋盘样式：传统、现代、木纹
- 支持深色/浅色主题
- 棋盘翻转功能
- 显示可移动位置提示
- 显示最后一步高亮

#### 引擎支持
- UCI 协议引擎支持
- 内置 Pikafish 引擎自动下载和配置
- 支持 Fairy-Stockfish 等多引擎
- 引擎选项配置（Hash、Threads、Skill Level 等）
- 分析模式支持
- 多引擎配置支持

#### 棋谱管理
- PGN 格式导入导出
- 自定义格式保存
- 复制/粘贴棋谱（支持剪贴板）
- 复制/粘贴局面（FEN 格式）
- 棋谱元数据编辑
- 最近文件列表

#### 时间控制
- 支持多种时间模式
- 自定义时间设置
- 显示双方用时

#### 键盘快捷键
- 丰富的快捷键支持
- 支持快捷键自定义
- 无鼠标操作支持

#### 视图控制
- 全屏模式支持
- 界面缩放功能
- 面板显示/隐藏控制
- 窗口布局保存和加载

### 技术特性

- 使用 SwiftUI 和 AppKit 构建
- 支持 macOS 14.0 及更高版本
- 支持 Apple Silicon 和 Intel Mac
- MVVM 架构设计
- 响应式界面设计

### 文档

- 完整的用户文档
- 安装指南
- 快速开始教程
- 用户手册

## 版本历史格式说明

### 版本号规则

遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号**：做了不兼容的 API 修改
- **次版本号**：做了向下兼容的功能性新增
- **修订号**：做了向下兼容的问题修正

### 变更类型

- **Added**: 新功能
- **Changed**: 对现有功能的变更
- **Deprecated**: 已经弃用的功能
- **Removed**: 已经移除的功能
- **Fixed**: 问题修复
- **Security**: 安全相关的修复

## 如何获取更新

### 自动更新

应用支持自动检查更新：

1. 点击菜单 **帮助** > **检查更新**
2. 如果有新版本，会提示下载
3. 下载完成后自动安装

### 手动更新

1. 访问 [Releases](../../releases) 页面
2. 下载最新版本的 DMG 文件
3. 按照 [安装指南](docs/Installation.md) 进行安装

### 查看当前版本

- 菜单 **中国象棋** > **关于中国象棋**
- 或按 `Cmd+?`

**注意**: 本应用持续更新中，建议保持更新到最新版本以获得最佳体验和最新功能。
