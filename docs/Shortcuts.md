# 键盘快捷键参考

本文档列出中国象棋 macOS 应用的所有键盘快捷键。

## 快捷键表

### 文件操作

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 新局 | `Cmd+N` | 开始新对局 |
| 打开 | `Cmd+O` | 打开棋谱文件 |
| 保存 | `Cmd+S` | 保存当前棋谱 |
| 另存为 | `Cmd+Shift+S` | 保存为新文件 |
| 导出 PGN | - | 导出为标准 PGN 格式 |
| 打印 | `Cmd+P` | 打印当前局面 |
| 退出 | `Cmd+Q` | 退出应用 |

### 编辑操作

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 悔棋 | `Cmd+Z` | 撤销上一步 |
| 重做 | `Cmd+Shift+Z` | 恢复撤销的着法 |
| 剪切 | `Cmd+X` | 剪切选中内容 |
| 复制 | `Cmd+C` | 复制选中内容 |
| 粘贴 | `Cmd+V` | 粘贴内容 |
| 全选 | `Cmd+A` | 全选 |
| 复制局面 | `Cmd+Shift+C` | 复制 FEN 局面 |
| 粘贴局面 | `Cmd+Shift+V` | 从 FEN 粘贴局面 |
| 复制棋谱 | `Cmd+Option+C` | 复制 PGN 棋谱 |
| 粘贴棋谱 | `Cmd+Option+V` | 从 PGN 粘贴棋谱 |
| 首选项 | `Cmd+,` | 打开设置 |

## 快捷键自定义

如果默认快捷键与其他应用冲突，可以自定义：

1. 打开 **首选项**（`Cmd+,`）
2. 选择 **快捷键** 标签页
3. 找到要修改的快捷键
4. 点击当前快捷键
5. 按下新的组合键
6. 点击"保存"

### 恢复默认快捷键

如果自定义快捷键后出现问题：

1. 打开 **首选项** > **快捷键**
2. 点击"恢复默认"按钮
3. 确认恢复

## 键盘操作技巧

### 无鼠标操作

熟练后可以实现完全键盘操作：

1. **打开棋谱**: `Cmd+O`，使用方向键选择文件，回车打开
2. **浏览棋谱**: 使用 `←` `→` 前后翻步
3. **悔棋/重做**: `Cmd+Z` / `Cmd+Shift+Z`
4. **保存棋谱**: `Cmd+S`
5. **退出**: `Cmd+Q`

### 快速导航

- `Cmd+Up` / `Cmd+Down`: 快速跳转到对局开始/结束
- `Left` / `Right`: 逐步浏览着法
- `Space`: 暂停/继续分析
