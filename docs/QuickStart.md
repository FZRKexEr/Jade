## 启动应用

### 首次启动

1. 在 Applications 文件夹中找到 **中国象棋** 应用
2. 双击图标启动

<!-- TODO: Screenshot - 应用图标截图 -->

3. 如果是首次启动，可能会看到安全警告，请按照 [安装指南](Installation.md) 中的说明允许应用运行

### 欢迎界面

首次启动时会显示欢迎界面：

<!-- TODO: Screenshot - 欢迎界面截图 -->

1. **选择语言**: 选择界面语言（简体中文/繁体中文/English）
2. **选择主题**: 选择深色或浅色主题
3. **选择棋盘样式**: 选择你喜欢的棋盘外观

点击"开始使用"进入主界面。

### 主界面介绍

<!-- TODO: Screenshot - 主界面标注截图 -->

主界面包含以下区域：

| 区域 | 说明 |
|------|------|
| 菜单栏 | 文件、编辑、视图、引擎、帮助等菜单 |
| 棋盘区 | 中央的中国象棋棋盘 |

| 着法列表 | 显示已走的棋步记录 |

## 第一盘棋

### 新局设置

1. 点击菜单 **文件 > 新局** 或按 `Cmd+N`

<!-- TODO: Screenshot - 新局对话框截图 -->

2. 在弹出的对话框中设置：
   - **红方**: 选择"人类"或"引擎"
   - **黑方**: 选择"人类"或"引擎"
   - **难度**: 如果选择了引擎，可以设置难度等级（1-20）

3. 点击"开始"按钮

### 人机对战示例

让我们开始第一盘人机对战：

1. **设置红方为人类，黑方为引擎**
2. **难度选择 5**（中等难度）
3. 点击"开始"

<!-- TODO: Screenshot - 新局设置截图 -->

### 走子操作

支持两种走子方式：

#### 方式一：点击走子

1. 点击要移动的棋子，该棋子会被高亮显示
2. 点击目标位置，如果走法合法，棋子将移动到该位置

#### 方式二：拖拽走子

1. 按住要移动的棋子
2. 拖动到目标位置
3. 释放鼠标完成走子

<!-- TODO: Screenshot - 走子操作截图 -->

### 着法显示

每走一步，着法会显示在右侧的着法列表中：

| 列 | 含义 |
|---|------|
| 步数 | 第几步棋 |
| 红方 | 红方的着法 |
| 黑方 | 黑方的着法 |

例如：
```
1. 炮二平五 马8进7
2. 马二进三 车9平8
```

## 调整难度设置

### 实时调整难度

在对局过程中，你可以随时调整引擎难度：

1. 点击菜单 **引擎 > 引擎选项**

<!-- TODO: Screenshot - 引擎选项对话框截图 -->

2. 调整以下参数：
   - **Skill Level**: 1-20，数值越大引擎越强
   - **Threads**: 引擎使用的 CPU 线程数
   - **Hash**: 引擎使用的内存大小（MB）

3. 点击"应用"保存设置

### 难度等级参考

| Skill Level | 适合人群 | 引擎强度 |
|-------------|----------|----------|
| 1-3 | 初学者 | 初学者水平 |
| 4-7 | 业余爱好者 | 业余初级 |
| 8-12 | 进阶玩家 | 业余中级 |
| 13-17 | 高手 | 业余高级 |
| 18-20 | 专业棋手 | 专业级 |

## 使用提示功能

当你不确定如何走时，可以向引擎请求提示：

### 请求提示

1. 确保轮到你的回合
2. 点击工具栏的 **提示** 按钮（灯泡图标）
3. 或按快捷键 `Cmd+H`

<!-- TODO: Screenshot - 提示功能截图 -->

### 提示显示

引擎会分析当前局面并显示：

- **推荐着法**: 当前最佳走法
- **评估分数**: 局面对红方的评估值（正数表示红方优势）
- **思考线**: 引擎计算的后续着法序列

例如：
```
推荐着法: 炮二平五
评估: +0.35
分析线: 1. 炮二平五 马8进7 2. 马二进三 车9平8 ...
```

## 悔棋和重做

### 悔棋

当你走错或想重新考虑时，可以悔棋：

**方法 1**: 点击工具栏的 **悔棋** 按钮（左箭头）
**方法 2**: 按快捷键 `Cmd+Z`
**方法 3**: 点击菜单 **编辑 > 悔棋**

### 重做

悔棋后可以重做：

**方法 1**: 点击工具栏的 **重做** 按钮（右箭头）
**方法 2**: 按快捷键 `Cmd+Shift+Z` 或 `Cmd+Y`
**方法 3**: 点击菜单 **编辑 > 重做**

### 悔棋规则

- 人机对战中：可以悔任意步数
- 人人对战中：需要双方同意悔棋
- 已保存并退出的棋局：重新加载后不能悔棋

## 保存和加载棋谱

### 保存棋谱

对局过程中或结束后，可以保存棋谱：

1. 点击菜单 **文件 > 保存** 或按 `Cmd+S`

2. 在保存对话框中：
   - 选择保存位置
   - 输入文件名
   - 选择格式（PGN 或 自定义格式）

3. 填写棋谱信息（可选）：
   - 对局日期
   - 红方/黑方姓名
   - 比赛名称
   - 对局结果

4. 点击"保存"

<!-- TODO: Screenshot - 保存棋谱对话框截图 -->

### 加载棋谱

可以加载已有的棋谱文件：

1. 点击菜单 **文件 > 打开** 或按 `Cmd+O`

2. 在文件选择器中：
   - 导航到棋谱文件位置
   - 选择 `.pgn` 或 `.xiangqi` 文件

3. 点击"打开"

4. 棋谱加载后：
   - 棋盘显示初始局面
   - 着法列表显示所有着法
   - 可以使用悔棋/重做浏览棋谱

### 棋谱格式支持

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| PGN | `.pgn` | 国际通用棋谱格式，兼容性最好 |
| 自定义 | `.xiangqi` | 应用专用格式，支持更多元数据 |

### 从剪贴板导入/导出

支持快速复制和粘贴棋谱：

**导出到剪贴板**:
1. 点击菜单 **编辑 > 复制棋谱**
2. 或使用快捷键 `Cmd+Shift+C`
3. PGN 格式的棋谱现在在你的剪贴板中

**从剪贴板导入**:
1. 复制 PGN 格式的棋谱内容
2. 点击菜单 **编辑 > 粘贴棋谱**
3. 或使用快捷键 `Cmd+Shift+V`
4. 棋谱会自动加载到应用中
